# 会计

## 会计公式

资产=负债+所有者权益

利润=收入-费用

资产=负债+所有者权益+（收入-费用）=负债+所有者权益+利润

## 余额方向

| 会计科目类别 | 借方       | 贷方       | 方向       |
| ------------ | ---------- | ---------- | ---------- |
| 资产         | 增加       | 减少       | 借方       |
| 负债         | 减少       | 增加       | 贷方       |
| 所有者权益   | 减少       | 增加       | 贷方       |
| 收入         | 减少或转销 | 增加       | 一般无余额 |
| 费用         | 增加       | 减少或转销 | 一般无余额 |
| 成本         | 增加       | 减少或转销 | 借方       |
